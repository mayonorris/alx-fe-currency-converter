# 💱 Currency Converter

A modern, responsive currency converter web application built with React, Vite, and Tailwind CSS.

This project is part of my ALX Front-End Capstone Project, designed to demonstrate API integration, state management, and responsive UI development in a real-world scenario.

_(Will be added after deployment)_

## Project Overview

- The Currency Converter application allows users to:

- Convert amounts between different currencies

- Fetch real-time exchange rates from a public currency API

- Automatically update conversion results

- Save favorite currency pairs for quick access

The project simulates a real-world fintech tool and focuses on usability, clean architecture, and responsive design.

## Features

> **Core Features**

- Real-time exchange rate fetching

- Automatic currency conversion

- Currency swap functionality

- Responsive UI (desktop, tablet, mobile)

- Error handling and loading states

> **MVP+ Feature**

- Save favorite currency pairs (stored locally)

- Planned Stretch Features

- Conversion history

> **Historical exchange rate chart**

- Rate variation indicators (DoD, WoW, MoM, YoY)

- Dark mode

## Tech Stack

- React

- Vite

- Tailwind CSS

- JavaScript (ES6+)

- Public Currency API : **ExchangeRate-API**

## Learning Objectives

- This project demonstrates:

- API integration using fetch

- Managing asynchronous data in React

- Using useState and useEffect

- Local storage management

- Component-based architecture

- Responsive design using Tailwind CSS
